What is Job Manager?

This is a site to host utility functions as a service for Caltrans Asset Management. Registered uer can the submit a job from a list of available functions. User can view submitted jobs, cancel jobs, download the output file if the job is completed.

Click the Function link to see the list of functions.

For registered user, click the New Job link to submit a new job.
